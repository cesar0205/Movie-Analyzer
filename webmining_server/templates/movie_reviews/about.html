Movie Sentiment Analyzer is a project that analyzes the sentiment of the reviews on a given movie title and ranks each review page using a basic version of the PageRank algorithm.

Sentiment analysis

  1. By querying a movie title the engine makes an automated query to Google or Bing.
  2. It saves the urls of the search results and a spider collects the reviews of each page.
  3. A Multinomial Naive Bayes Classifier is used to analyse the reviews and classify them into positive or negative.
  4. A graphic is displayed with the classification results.

PG Rank

  1. After the sentiment analysis results are displayed, the user can rank the movie review pages by clicking "Calculate Rank".
  2. A crawler gathers and travels through each of the review urls with a max depth of 2 and saves the pages and creates links between the visited pages.
  3. The engine used the PageRank algorithm to rank each page in the collected page set.

This engine is written in Python, Scikit-Learn and NLTK on top of NumPy and SciPy stack. It uses Django for webserver backend.
